Azure Windows Virtual Desktop (WVD)

  • Работа из любого места и на любом устройстве
  • Развертывание и масштабирование в считанные минуты
  • Снижение затрат и модернизация
  • Защита пользовательских данных
  • Мульти-сессионность
  • Управление Win 7 End of Support с помощью WVD
  • Виртуализация приложений Office 365
02

Обеспечение удаленного
доступа к приложениям
в локальной среде

Azure Application Proxy

  • Простота в использовании: не нужно обновлять или изменять приложения для работы с Azure Application Proxy
  • Безопасность: локальные приложения могут использовать аналитику безопасности Azure и двухфакторную проверку подлинности
  • Экономичность: не нужно изменять сетевую инфраструктуру или устанавливать дополнительные устройства в локальной среде
  • Удаленный доступ с низкой нагрузкой на ИТ-инфраструктуру

Серверные решения

Azure Virtual Machine

  • Новый сервер для бизнес-приложений
  • Модернизация сервера, повышение его производительности
  • Замена устаревшего оборудования
  • Расширение серверного парка, увеличение вычислительных мощностей

Azure Site Recovery

  • Резервный сервер
  • Обеспечение отказоустойчивости важных сервисов

Azure Backup

  • Резервное копирование
